# GTK Proton Prefix Manager


### Supported features:
1. Steam games.
2. Non Steam games added to steam.

### Dependencies
Arch:

```bash
sudo pacman -S python-gobject gtk4 libadwaita gobject-introspection
```
Fedora:

```bash
sudo dnf install python3-gobject gtk4 libadwaita
```

Debian/Ubuntu:
```bash
sudo apt install python3-gi python3-gi-cairo gir1.2-gtk-4.0 libadwaita-1-0
```
### Running

```bash
git clone https://github.com/mhashirshahzad/gtk-proton-save-mgr

cd gtk-proton-save-mgr

chmod +x run.sh

./run.sh
```
